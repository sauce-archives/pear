saucelabs's PEAR Channel Server

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover saucelabs.github.com/pear
Listing available packages: 
pear remote-list -c saucelabs
Installing a package: 
pear install saucelabs/package_name
Installing a specific version/stability: 
pear install saucelabs/package_name-1.0.0
pear install saucelabs/package_name-beta
Receiving updates via a feed: 
http://saucelabs.github.com/pear/feed.xml
